Computer Science I

This course provided a rigorous introduction to the fundamental concepts and techniques of computer programming using an object-oriented language. Key topics included:

  • Data Abstraction: Modeling complex data through abstract data types.
  • Recursion: Solving problems using recursive methods.
  • Generic Data Types: Programming with flexible and reusable data structures.
  • Inheritance and Polymorphism: Applying these core object-oriented principles to enhance code reusability and adaptability.
  • Exception Handling: Effectively managing errors to ensure robust software development.

The concepts learned were applied to developing console programs and engaging in event-driven programming using a simple graphics API.

Computer Science II

This course focused on the design and implementation of fundamental abstract data types in computer science, including stacks, queues, trees, lists, hashing, and graphs, using an object-oriented programming language. Key programming requirements included:

  • Implementation of Abstract Data Types: Utilizing arrays and dynamic links.
  • Recursion: Employing recursive techniques for problem-solving.
  • Sorting and Searching: Understanding various algorithms for data organization and retrieval.
  • Hashing: Learning efficient data storage and access methods.
  • String Processing: Manipulating and analyzing string data effectively.

Additionally, the course introduced students to algorithm analysis, equipping them with the skills to evaluate the efficiency and performance of algorithms.

Computer Networking

This course provides a comprehensive overview of computer networking with no prior networking knowledge required. It employs a top-to-bottom approach to explore essential network concepts and protocols, covering:

Key Topics:

  • High-Level Applications: Understanding how applications utilize network protocols.
  • Core Protocols: In-depth study of key protocols such as HTTP, TCP/IP, and routing.
  • Network Types: Exploration of Local Area Networks (LANs), wireless networking, and other configurations.
  • Socket Programming: Gaining hands-on experience in data communication through socket programming.

By the end of the course, students will have a solid understanding of both theoretical concepts and practical applications in computer networking.

Principles of Web Technology

This course covers the history and development of the World Wide Web, providing a comprehensive overview of the technologies and methodologies that shape modern web design and development. Key topics include:

Key Topics:

  • History of the Web: Exploring the evolution of the World Wide Web and its impact on society.
  • Content Authoring: Techniques for creating both static and dynamic content for web applications.
  • Web Security: Learning best practices for ensuring the security of web applications and protecting user data.
  • Electronic Commerce: Understanding the principles of e-commerce and the technologies that facilitate online transactions.

Additionally, students will engage in a significant group project that allows them to apply these concepts in a practical setting.

Systems Administration

This course focuses on the deployment and maintenance of modern computer systems in an operational environment, providing both conceptual knowledge and practical experience. Key topics covered include:

Key Topics:

  • Architectures: Understanding various system architectures and their components.
  • Heterogeneous Systems: Managing diverse systems that operate on different platforms.
  • Authentication and Security: Implementing robust security measures and authentication protocols.
  • Network Services: Exploring essential network services, including firewalls and storage solutions.
  • Performance Analysis and Tuning: Techniques for analyzing system performance and optimizing configurations.
  • Management and Configuration: Best practices for managing services and system resources effectively.
  • System Initialization: Understanding the boot process and system initialization procedures.
  • Drivers: Working with device drivers for hardware compatibility.
  • Cross-Platform Services: Configuring services that operate across multiple operating systems.
  • Policies and Procedures: Establishing operational policies and procedures for effective system administration.

By the end of the course, students will have developed the skills necessary to maintain and manage modern computer systems efficiently.

Introduction to Programming (Java)

This course provided an introduction to the fundamentals of programming using Java, covering key topics such as:

  • Object-Oriented Programming: Implementing OOP principles to create modular and reusable code.
  • Classes and Methods: Developing classes and methods to define objects and their behaviors.
  • Control Structures: Utilizing flow control statements like loops and conditionals to manage program execution.
  • Data Types and Inheritance: Understanding various data types and using inheritance to promote code reusability.
  • Exception Handling and File I/O: Managing errors effectively and performing file input/output operations.

Students gained practical experience in developing Java applications.

Calculus I and II

These courses encompass the study of change and the fundamental concepts of calculus, covering:

Calculus I:

  • Understanding definitions, concepts, and interpretations of the derivative and integrals (definite and indefinite).
  • Topics include differentiation, integration, and their applications.
  • Review of the definition of the definite integral and the Fundamental Theorem of Calculus.

Calculus II:

  • Techniques of integration, including approximate integration and improper integrals.
  • Applications of integration in various fields.
  • Exploration of sequences and series, and approximating functions using polynomials and Taylor series.

Together, these courses provide a solid foundation in calculus principles and their practical applications.

Reliability and Engineering Statistics

This course provides a comprehensive overview of statistical concepts and techniques, including:

  • Probability Distributions: Understanding hypergeometric, binomial, Poisson, and normal probability distributions.
  • Confidence Intervals: Constructing and interpreting confidence intervals for population parameters.
  • Inferences Concerning Means: Making statistical inferences related to population means.
  • Linear Regression: Exploring relationships between variables through linear regression analysis.
  • Statistical Quality Control and Reliability: Introduction to concepts in quality control and reliability assessment.
  • Use of Computers: Applying statistical software for data analysis and visualization.

Students will gain practical experience in statistical methods and their applications in various fields.
